# 🚀 Go Avatar API

A high-performance backend service built with **Go** that generates unique user avatars. This project demonstrates how to handle RESTful routing, external API integration, and dynamic string manipulation.

## 🌟 Features
- **Human Avatars**: Integrated with **Gravatar** using MD5 hashing.
- **Robot & Pixel Avatars**: Powered by **DiceBear**, allowing for millions of unique combinations.
- **Dynamic Styling**: Support for custom hair, clothing, and sizing via URL parameters.
- **Fast & Lightweight**: Built using the **Gin-Gonic** framework.

## 🛠️ Tech Stack
- **Language:** Go (Golang)
- **Framework:** Gin-Gonic
- **Environment:** Ubuntu Linux / VS Code

## 🚀 Getting Started

### Prerequisites
- Go 1.20 or higher installed on your Ubuntu machine.
- Git for version control.
## How to run
1. Clone the repo
2. Run `go run .`/ `go run main.go`
3. Visit `http://localhost:8080/profile?email=test@test.com`
