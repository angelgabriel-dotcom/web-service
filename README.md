# Go Avatar API
This is a REST API built with **Gin** that generates custom avatars.

## Features
- Generates Gravatar links from email hashes.
- Creates custom robots and characters using DiceBear.
- Built with Golang.

## How to run
1. Clone the repo
2. Run `go run .`/ `go run main.go`
3. Visit `http://localhost:8080/profile?email=test@test.com`
